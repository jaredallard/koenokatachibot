# Koe no Katachi Bot

Post movies on ~~Twitter~~ Mastodon in screenshot form for others to enjoy / consume.

## Usage

Configure the bot via `config/config.example.json`, save that as `config/config.json`.

```bash
$ cargo run 

# Or, with Docker
$ ./docker_start.sh
$ docker logs -f <id_dumped_out>
```
